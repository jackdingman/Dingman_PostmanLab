Postman Collection JSON file is located in SRC -> Main > resources.

Once you have downloaded the entire project, run the StudentRegDemo file within
  src->main->java->com.baarschbytes->studentRegDemo->validation

Run the file to start the Arkansas Student API.

Next, import the Postman Collection JSON file into Postman. Do this by taking the following steps:
  Click File (top left corner)
  Click Import
  Select the JSON File
  Click the small three dots next to the collection title (View More Actions)
  Click Run
  Click Start Run
  
Test Suite is also located within the resources folder as an excel file.
